
# FocusForge

FocusForge is a Chrome Extension designed to help students and professionals stay focused and productive.

## Features

* Blocks distracting websites
* Filters non-study YouTube content
* Focus Score tracking
* Productivity analytics dashboard
* Voice-based feedback
* Achievement badge system
* Real-time activity monitoring

## Technology Stack

* React
* TypeScript
* JavaScript
* HTML
* CSS
* Vite
* Chrome Extension APIs

## Purpose

The goal of FocusForge is to reduce distractions, improve productivity, and help users build better study and work habits through intelligent focus tracking and analytics.
